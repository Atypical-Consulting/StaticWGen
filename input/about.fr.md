# À propos de cette démo

Bienvenue sur le site de démonstration ! Ce site est généré avec **Nuke**, un outil d'automatisation de build, et **Pico CSS** pour le style.

## Fonctionnalités

- Conversion de Markdown en HTML
- Coloration syntaxique avec Prism.js
- Diagrammes Mermaid
- Flux RSS/Atom
- Système de tags
